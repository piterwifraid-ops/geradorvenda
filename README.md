# STSTS - React + Tailwind CSS

Um projeto React com Vite e Tailwind CSS, apresentando uma tela em branco pronta para customização.

## Requisitos

- Node.js 16+ 
- npm ou yarn

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

A aplicação estará disponível em `http://localhost:5173`

## Construção

```bash
npm run build
```

## Pré-visualização

```bash
npm run preview
```

## Estrutura do Projeto

```
├── src/
│   ├── App.jsx        # Componente principal
│   ├── main.jsx       # Ponto de entrada
│   └── index.css      # Estilos globais
├── index.html         # HTML principal
├── package.json       # Dependências
├── vite.config.js     # Configuração Vite
└── tailwind.config.js # Configuração Tailwind
```

## Personalização

Edite o arquivo `src/App.jsx` para adicionar seu conteúdo à página em branco.
